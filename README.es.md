# Code Vision

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad de Code Vision para OpenClaw.

## Lo que soluciona
Visualiza la estructura del código a través de gráficos interactivos y mapeo de dependencias.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales y listos para producción
- Enfoque de seguridad primero
- Soporte para revertir cambios

## Uso
/code-vision

## Ejemplo
```
Analizar el código para identificar problemas de diseño.
```

## Instalación
Esta habilidad está automáticamente disponible en OpenClaw.
